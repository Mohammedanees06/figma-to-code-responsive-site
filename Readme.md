# Industrial Product Website – Responsive UI Implementation

## 📌 Overview

This project is a fully responsive industrial product website built using vanilla HTML, CSS, and JavaScript.  
The goal was to convert a detailed Figma design into a pixel-accurate, interactive frontend implementation.

The project focuses on clean structure, modern layout systems, and smooth UI interactions without using any external frameworks or libraries.

---

## 🚀 Features

- Fully responsive layout (Desktop, Tablet, Mobile)
- Sticky header triggered on scroll
- Interactive image carousel with smooth transitions
- Hover-based zoom preview
- Accordion-style FAQ section
- Tab-based content switching
- Structured multi-section landing page
- Semantic HTML5 structure
- Modern CSS (Flexbox & Grid)
- Clean separation of concerns (HTML / CSS / JS)

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript

No frameworks or external UI libraries were used.

## 🎯 Key Implementation Details

### Responsive Design
- CSS Grid used for structured layout sections
- Flexbox used for alignment and distribution
- Media queries for breakpoint handling

### Sticky Header
- Scroll-based detection logic
- Smooth transition animation
- Fixed positioning without layout shift

### Carousel
- `transform: translateX()` for smooth GPU-accelerated movement
- State-based index management
- Overflow handling for slide containment

### Accessibility Considerations
- Semantic elements
- Button-based interactions
- Clear visual hierarchy
